# 📋 История изменений

- CPU-физика (fallback для старых ПК)
- Звук (XAudio2, WAV)
- Загрузка OBJ-моделей
- Демо-сцена: 81 падающий куб

- Прыжок теперь срабатывает мгновенно
- Alignment структур для GPU
- Ошибки в матрице rotationY

- Texture::loadFromFile — заглушка
- GPU-физика требует OpenGL 4.3+
