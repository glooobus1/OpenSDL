# 📋 История изменений

## [1.0.0] - 2025-04-10

### ✨ Добавлено

#### Рендеринг и графика
- Полноценный 3D-рендеринг на OpenGL 3.3+
- Камера от первого лица (WASD + мышь)
- Тени (shadow mapping) с возможностью отключения
- Улучшенное освещение: Phong + specular блики
- Мягкие тени (PCF 2x2/3x3)
- Встроенные примитивы: куб, сфера, плоскость
- Загрузка OBJ-моделей (Wavefront .obj)
- Рендер текста (встроенный растровый шрифт 8x8)
- Видимая сфера солнца (источник света + визуальный объект)
- Цвет неба (настраивается через `WORLD_CLEAR_COLOR`)

#### Физика
- GPU-физика через Compute Shader (SSBO, OpenGL 4.3+)
- CPU-физика как fallback (совместимость с OpenGL 3.3)
- Гравитация (настраивается)
- Коллизии AABB и сфера
- Restitution (упругость) и friction (трение)

#### GPU-ускорение
- GPU-буферы для матриц трансформации (SSBO, binding 0 и 1)
- GPU-физика (SSBO, binding 2, compute shader)
- Увеличен буфер объектов до 5000-10000

#### Управление
- Управление камерой: WASD + мышь
- Прыжок (мгновенное срабатывание, исправлено)
- Инерция движения
- Покачивание камеры при ходьбе
- Геймпад (SDL3 Gamepad)
- Инверсия осей мыши (X/Y)

#### Звук
- Звуковая система на XAudio2
- Загрузка и воспроизведение WAV
- Поддержка loop и громкости

#### Прочее
- Своя математическая библиотека (`vec2/3/4`, `mat4`, `quat`)
- Настройки движка вынесены в публичные переменные
- Тест производительности: FPS в заголовке окна
- Демо-сцена: сетка из 81 падающего куба (9x9)

### 🔧 Изменения в архитектуре
- `Camera.hpp` разделена на `Camera.hpp` и `StandartPlayerController.hpp`
- Улучшен alignment структур для GPU (`vec3` → 16 байт с паддингами)
- Оптимизирована работа с матрицами (кэширование, dirty-флаги)

### 🐛 Исправлено
- Прыжок: теперь срабатывает мгновенно, а не по зажатию
- Alignment структур для GPU-физики (`GPURigidbody`)
- Ошибки в матрице поворота `rotationY`
- Утечки памяти в шейдерах и GPU-буферах

### ⚠️ Известные проблемы
- `Texture::loadFromFile` — заглушка (нужно реализовать через stb_image)
- OBJ-загрузчик работает только с треугольниками (не квады)
- GPU-физика требует OpenGL 4.3+ (на старых картах отключать флагом)
