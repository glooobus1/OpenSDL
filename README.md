\# 🎮 OpenSDL Game Engine



\*\*OpenSDL\*\* — это легковесный игровой движок на C++ с OpenGL 3.3/4.3, физикой, тенями, звуком и поддержкой GPU-вычислений.



\## ✨ Возможности



\- 🖼️ \*\*Рендеринг\*\* на OpenGL 3.3+ (шейдеры, текстуры, материалы)

\- 💡 \*\*Динамическое освещение\*\* + карты теней (shadow mapping)

\- ⚙️ \*\*Физика твёрдых тел\*\* (гравитация, коллизии AABB/сфера)

\- 🚀 \*\*GPU-физика\*\* через Compute Shaders (SSBO, OpenGL 4.3+)

\- 🎮 \*\*Управление\*\* (клавиатура + мышь + геймпад SDL3)

\- 🔊 \*\*Звук\*\* через XAudio2 (WAV)

\- 📦 \*\*Загрузка OBJ-моделей\*\* (Wavefront .obj)

\- 🧪 \*\*Встроенные примитивы\*\*: куб, сфера, плоскость, солнце

\- 📝 \*\*Рендер текста\*\* (встроенный растровый шрифт 8x8)



\## 🕹️ Демо-сцена



После запуска вы увидите:

\- Плоскость (земля) с зелёной травой

\- Сетку из 81 куба (9x9), падающих под гравитацией

\- Солнце (источник света + визуальная сфера)

\- Тени (если включены)

\- Фоновую музыку (если файл найден)



\## 🎮 Управление



| Клавиша | Действие |

|---------|----------|

| `W` / `S` | Движение вперёд/назад |

| `A` / `D` | Движение влево/вправо |

| `Пробел` | Прыжок |

| `Мышь` | Поворот камеры |

| `ПКМ` | Захватить мышь |

| `ESC` | Отпустить мышь |



\## 📁 Быстрый старт



1\. Открой `OpenSDL.vcxproj` в Visual Studio 2022

2\. Выбери `Release | x64`

3\. Нажми `Build Solution` (Ctrl+Shift+B)

4\. Запусти (Ctrl+F5)



Подробнее о сборке — в \[`BUILD.md`](BUILD.md)
