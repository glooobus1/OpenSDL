# OpenSDL Game Engine

OpenSDL — это лёгкий 3D-движок на C++ и OpenGL.  
Создан для авиасимулятора и оптимизирован для работы на школьных ПК.

---

## Возможности

**Графика**
- OpenGL 3.3 / 4.3 (автоматический выбор)
- GPU-матрицы (SSBO) — до 10 000 объектов
- Phong освещение + specular блики
- Мягкие тени (PCF 2x2 / 3x3)
- Видимое солнце (сфера)
- Текстурное небо (SkyTexture)

**Физика**
- CPU-физика (стабильная, для школьных ПК)
- GPU-физика (compute shader, для мощных ПК)
- Гравитация, прыжки, коллизии AABB
- Инерция и покачивание камеры

**Звук**
- XAudio2 (WAV-файлы)
- Поддержка петель и громкости

**Управление**
- WASD + мышь (от первого лица)
- Прыжок (пробел)
- Захват мыши (ПКМ) / отпускание (ESC)
- Поддержка геймпада

**Производительность**
- 1000+ FPS на простых сценах
- 60-80 FPS на 2500 кубах с физикой
- 30 FPS на 10 000 кубах без физики

---

## Демо-сцена

После запуска вы увидите:

- Плоскость (земля) с зелёной травой
- Сетку из 81 куба (9x9), падающих под гравитацией
- Солнце (источник света + видимая жёлтая сфера)
- Тени (если включены в настройках)
- Фоновую музыку (если файл найден)

---

## Управление

- `WASD` — движение
- `Мышь` — поворот камеры
- `Пробел` — прыжок
- `ПКМ` — захватить мышь
- `ESC` — отпустить мышь

---

## Сборка

**Требования**
- Visual Studio 2022
- SDL3
- GLEW
- OpenGL 3.3 / 4.3
- XAudio2 (Windows)

**Шаги**
1. Открой `OpenSDL.sln`
2. Выбери `Release | x64`
3. Нажми `Ctrl+Shift+B`
4. Запусти `Ctrl+F5`

---

## Настройки

Все настройки в файле `StandartOpenGlRender.hpp`:

- `ENABLE_SHADOWS` — тени (на слабых ПК выключить)
- `ENABLE_VSYNC` — вертикальная синхронизация
- `ENABLE_GPU_MATRICES` — GPU-матрицы (OpenGL 4.3+)
- `ENABLE_GPU_PHYSICS` — GPU-физика (экспериментально)
- `MAX_GPU_OBJECTS` — максимум объектов (по умолчанию 5000)
- `GRAVITY` — вектор гравитации
- `PLAYER_MOVE_SPEED` — скорость игрока
- `PLAYER_JUMP_POWER` — сила прыжка
- `CAMERA_FOV` — поле зрения

Подробнее — в файле `Settings.md`.
