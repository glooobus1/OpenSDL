\# ⚙️ Настройки движка OpenSDL



Все настройки находятся в файле \*\*`StandartOpenGlRender.hpp`\*\* в публичных переменных класса `StandartOpenGlRender`.



\## 🎨 Графика



| Переменная | Тип | Значение | Описание |

|------------|-----|----------|----------|

| `ENABLE\_SHADOWS` | `bool` | `true` | Включить карты теней |

| `ENABLE\_VSYNC` | `bool` | `false` | Вертикальная синхронизация |

| `ENABLE\_DEBUG\_COLLIDERS` | `bool` | `false` | Рисовать зелёные рамки коллайдеров |

| `ENABLE\_GPU\_MATRICES` | `bool` | `true` | Матрицы на GPU (требует OpenGL 4.3+) |

| `MAX\_GPU\_OBJECTS` | `int` | `5000` | Максимум объектов для GPU-матриц |

| `WORLD\_CLEAR\_COLOR` | `vec3` | `(0.5, 0.7, 1.0)` | Цвет неба |



\## ⚙️ Физика



| Переменная | Тип | Значение | Описание |

|------------|-----|----------|----------|

| `ENABLE\_GPU\_PHYSICS` | `bool` | `true` | Физика на GPU (OpenGL 4.3+) |

| `MAX\_PHYSICS\_OBJECTS` | `int` | `5000` | Максимум объектов для GPU-физики |

| `GRAVITY` | `vec3` | `(0, -9.8, 0)` | Вектор гравитации (м/с²) |



> При `ENABLE\_GPU\_PHYSICS = false` используется CPU-физика.



\## 🎮 Управление



| Переменная | Тип | Значение | Описание |

|------------|-----|----------|----------|

| `PLAYER\_MOVE\_SPEED` | `float` | `8.0` | Скорость движения (м/с) |

| `PLAYER\_MOUSE\_SENS` | `float` | `0.005` | Чувствительность мыши |

| `PLAYER\_JUMP\_POWER` | `float` | `5.5` | Сила прыжка (м/с) |

| `PLAYER\_GRAVITY` | `float` | `12.0` | Гравитация игрока (м/с²) |

| `INVERT\_X` | `bool` | `true` | Инвертировать мышь по X |

| `INVERT\_Y` | `bool` | `true` | Инвертировать мышь по Y |



\## 📷 Камера



| Переменная | Тип | Значение | Описание |

|------------|-----|----------|----------|

| `CAMERA\_FOV` | `float` | `80.0` | Поле зрения (градусы) |

| `CAMERA\_NEAR` | `float` | `0.1` | Ближняя плоскость |

| `CAMERA\_FAR` | `float` | `1000.0` | Дальняя плоскость |

| `CAMERA\_START\_POS` | `vec3` | `(0, 3, 15)` | Стартовая позиция |



\## 💡 Как изменить



В `Main.cpp` перед `render.init()`:



```cpp

StandartOpenGlRender render;

render.ENABLE\_SHADOWS = false;    // отключить тени

render.PLAYER\_MOVE\_SPEED = 15.0f; // быстрее бегать

render.init(800, 600, "My Game");

⚠️ Совместимость

OpenGL 4.3+ нужен для GPU-физики и GPU-матриц



На старых видеокартах установи флаги в false



Тени работают на OpenGL 3.3+
