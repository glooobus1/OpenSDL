\## 🐛 Сообщение об ошибках



Если нашёл баг:

1\. Проверь, не сообщали ли о нём ранее

2\. Создай Issue с описанием:

&#x20;  - Что делал

&#x20;  - Что ожидал

&#x20;  - Что произошло

&#x20;  - Скриншоты/логи



\## 💡 Предложение фич



Создай Issue с меткой `enhancement`:

\- Какая фича

\- Зачем нужна

\- Пример использования



\## 📝 Pull Request'ы



1\. Форкни репозиторий

2\. Создай ветку (`git checkout -b feature/amazing`)

3\. Внеси изменения

4\. Запушь (`git push origin feature/amazing`)

5\. Открой Pull Request в `main`



\### Требования к коду



\- Тот же стиль кода (отступы, имена)

\- Комментарии к сложным местам

\- Не ломай существующие API

\- Тестируй на OpenGL 3.3 и 4.3



\## 📚 Архитектура



\- `StandartOpenGlRender` — главный класс (рендер + физика)

\- `SceneObject` — базовый класс (transform + rigidbody + collider)

\- `Camera` — независимая камера

\- `PhysicsWorld` — CPU-физика (fallback)

\- `GPUPhysics` — GPU-физика (compute shader)



\## ❓ Вопросы



Создай Issue с меткой `question`
